# CivicPulse 🌆🤖
> **AI Powered Smart City Decision Intelligence Platform**
> **Built and Presented by LUKE JEBASUNDAR D**

CivicPulse is an advanced platform designed to transform chaotic urban data streams such as real time IoT sensor metrics and unstructured citizen feedback into proactive, AI driven administrative insights. 

This project was developed for the **Gen AI Academy APAC Edition Hackathon** under **Problem Statement 1 (PS1): AI-Powered Decision Intelligence Platform**.

---

## 🔗 Live Application
🚀 **View Live Project:** [https://civicpulse-gen-ai-hackathon.lovable.app](https://civicpulse-gen-ai-hackathon.lovable.app)

---

## 🛠️ Tech Stack
*   **Frontend UI:** React, Vite, Tailwind CSS (Built & Compiled via Lovable.ai)
*   **Cloud Backend:** Google Firebase & Cloud Firestore (Real-time Database Sync)
*   **Core AI Engine:** Google Vertex AI / Gemini 1.5 API (Reasoning & NLP Layer)
